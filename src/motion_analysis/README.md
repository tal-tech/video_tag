# dolphin motion analysis

## Dependencies
* PyTorch (>=1.1.0)
* NumPy
* OpenCV-Python
* scikit-video
* ...

## Installation
```bash
    cd ./lib/models/networks/DCNv2
    python setup.py build develop
```
